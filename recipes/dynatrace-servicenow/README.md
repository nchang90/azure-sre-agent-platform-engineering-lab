# dynatrace-servicenow

Upstream-style mirror for the Microsoft SRE Agent ServiceNow incident recipe.

This lab keeps the ServiceNow pieces in the same recipe layout the upstream repo uses:

- `agent.json`
- `connectors.json`
- `tool-permissions.json`
- `automations/incident-platforms/servicenow.yaml`
- `automations/incident-filters/snow-p1p2.yaml`

Populate the ServiceNow instance URL and credentials when importing the recipe.
